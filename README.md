# 🧮 Scientific Calculator (Python)

A simple command-line Scientific Calculator built using Python.  
This project performs basic arithmetic operations along with scientific functions.

---

## 🚀 Features
- Basic Operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division

- Scientific Functions:
  - Power (x^y)
  - Square Root
  - Logarithm (base 10)
  - Natural Log (ln)
  - Trigonometric functions (sin, cos, tan)

---

## 🛠️ Technologies Used
- Python 3
- Built-in `math` module
